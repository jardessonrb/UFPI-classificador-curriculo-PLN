Analista de dados
Sobre a vaga

Você deve ser analítico, ter pensamento crítico e habilidades com Data Viz, análise de resultados, desenvolvimento de conclusões e storytelling de dados para comunicar os insights descobertos para outras equipes e/ou clientes, orientando profissionais de outras áreas.Aqui na SX temos compromissos com a equidade de oportunidades para todas as pessoas. visualização de dados (Data Viz);- Analisar resultados de KPIs do negócio e performance das pessoas, identificando oportunidades de melhoria na entrega de resultados;- Criar e customizar reports Adhocs, desenvolver e manter dashboards que se façam necessários;- Participar de projetos de Ciência de Dados voltados para problemáticas das áreas de negócio que podem ser respondidas com dados;- utilizando técnicas estatísticas e de modelagem;- Interagir com múltiplos stakeholders para garantir entendimento adequado dos problemas e assegurar uso das soluções de Ciência de Dados nos
