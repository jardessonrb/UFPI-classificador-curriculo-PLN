RESPONSABILIDADES E ATRIBUIÇÕES
A área de Projetos, responsável por implantação e customização de projetos na área de fundos de investimentos, busca um(a) Desenvolvedor de Sistemas PL | Java para compor nosso time, na maior empresa de desenvolvimento de softwares e produtos para o mercado financeiro do Brasil, onde acreditamos que a inovação tecnológica inspira a evolução.
Esse cargo é responsável por:
• Ser um Desenvolvedor de Sistemas, com a principal atividade sendo o desenvolvimento em Java;
• Implementação (desenvolvimento e manutenção) de sistemas em Java, legado da empresa;
• Desenvolvimento de scripts de banco de dados MS SQL Server, necessários para melhorias e correções dos sistemas e realizar análise de performance e otimizações em consultas ao banco de dados;
• Participar de reuniões, configurar o ambiente de desenvolvimento local, documentar, realizar testes das aplicações em ambiente de desenvolvimento;
• Desenvolver código limpo, eficiente e de alta qualidade para atender aos requisitos do projeto;
• Colaborar com a equipe de desenvolvimento para planejar e executar tarefas;
• Realizar testes de unidade e integração para garantir a qualidade do software;
• Participar de reuniões de equipe para discutir o progresso do projeto e identificar possíveis problemas;
• Manter-se atualizado com as melhores práticas e tendências de desenvolvimento de software do time;
• Documentar o código, as evidências de teste e os processos relevantes para facilitar a manutenção futura;
• Escrever testes unitários e de integração para verificar se o código implementado está funcionando corretamente e atendendo aos critérios de aceitação.
