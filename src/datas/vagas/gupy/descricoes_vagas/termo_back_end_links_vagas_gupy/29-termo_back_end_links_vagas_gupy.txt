RESPONSABILIDADES E ATRIBUIÇÕES
Responsabilidades:
Desarrollar y mantener aplicaciones web responsivas utilizando HTML, CSS y JavaScript.
Colabore con diseñadores para traducir esquemas y maquetas en páginas web funcionales.
Integrar Liferay con otras aplicaciones y sistemas mediante APIs y web services.
Optimice las aplicaciones web para obtener la máxima velocidad y escalabilidad
Garantice la compatibilidad entre navegadores y resuelva cualquier problema que surja
Implementar y mantener frameworks y bibliotecas front-end.
Manténgase actualizado con las últimas tendencias y tecnologías en desarrollo front-end
Realizar revisiones de código y brindar comentarios constructivos a los miembros del equipo.
Solucionar y depurar problemas de front-end a medida que surjan
Colaborar con desarrolladores de back-end para integrar componentes de front-end con la lógica del lado del servidor.
REQUISITOS E QUALIFICAÇÕES
Requisitos:
Experiencia comprobada como desarrollador front-end o puesto similar.
Competente en HTML, CSS y JavaScript.
Experiencia con frameworks front-end como React, Angular o Vue.js.
Familiaridad con los sistemas de control de versiones (por ejemplo, Git)
Gran comprensión de los principios del diseño responsivo.
Conocimiento de los principios y mejores prácticas de SEO.
Excelentes habilidades de comunicación y resolución de problemas.
Capacidad para trabajar de forma independiente y como parte de un equipo.
Atención al detalle y pasión por crear interfaces visualmente atractivas y fáciles de usar.
Calificaciones preferidas:
Experiencia con diseño UI/UX.
Conocimiento de tecnologías back end como Node.js o PHP.
Familiaridad con metodologías de desarrollo ágiles.
