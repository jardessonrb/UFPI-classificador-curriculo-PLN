DESCRIÇÃO DA VAGA
A Ploomes é a maior empresa de CRM da América Latina.
Nosso objetivo é centralizar toda a operação comercial possibilitando a visão 360º dos clientes. 🌎
Para isso, desenvolvemos um produto que combina robustez e facilidade de implementação, utilização e manutenção, com vários módulos que vão auxiliar o dia a dia do time comercial.
Acreditamos na autonomia, na força das nossas conexões e na atitude que é a chave para gerar o impacto que queremos no mercado. 👊
Valorizamos um ambiente plural e diverso, por isso, candidaturas de pessoas pretas/negras, mulheres, indígenas, pessoas com deficiências, LGBTQIA+ ou outros grupos minorizados são muito bem vindas!
Inscreva-se e venha fazer parte do nosso time! 💜
Praticamente todo novo cliente da Ploomes possui demandas que são requisitos para que possamos atendê-lo, como por exemplo:
Integrar com sistemas terceiros
Automatizar processos
Customizar o comportamento do Ploomes
Para estas demandas, é necessário utilizar a API do Ploomes para desenvolver o projeto. Geralmente, quando se trata de Integrações, é necessário desenvolver um Middleware que seja capaz de:
Consultar a fonte (sistema 1) em busca de dados que possam ter sido atualizados
Validar se os dados encontrados estão dentro do critério definido para a integração
Tratar eventuais exceções
Transformar os dados (tipos, Identificadores, entre outros) para o formato esperado pelo destino (sistema 2)
Fazer o envio da informação para o destino
Para Automações de rotinas, por exemplo, é necessário criar middlewares ou usar Webhooks para integração do Ploomes com o próprio Ploomes, no estilo "If this then that". Ou seja, quando determinado evento acontecer (gatilho), um outro deve acontecer automaticamente. Exemplos reais:
Quando uma negociação atingir determinado estágio, uma tarefa de Follow Up deve ser criada ao vendedor responsável;
Quando um cliente ficar sem vendas realizadas por mais do que 3 meses, converte-lo em "inativo" e disparar uma notificação para o responsável pelo cliente;
Customizações, por sua vez, podem envolver:
Criar telas novas especificamente para um cliente, utilizando a API do Ploomes para criar novas capacidades para o sistema
Criar ferramentas de exportação de relatórios customizados

SOBRE NÓS
Muito prazer, somos a Ploomes: a maior empresa de CRM da América Latina! Prático, né? 🤗

Somos claros, objetivos e autênticos, garantindo que a outra pessoa entenda o que queremos comunicar. Através de encantamento, inspiração e energia, proporcionamos uma verdadeira experiência a nossos clientes, parceiros e nossos próprios colaboradores.
