DESCRIÇÃO DA VAGA
Estamos em busca de profissionais excepcionais para se juntarem ao nosso time de desenvolvimento para um projeto que está transformando o setor de marketing por meio de soluções tecnológicas inovadoras. Trabalhar conosco significa enfrentar desafios empolgantes, envolvendo arquiteturas modernas, automação de processos, e desenvolvimento de interfaces intuitivas que impactam diretamente nossos clientes. Utilizamos uma stack robusta, incluindo AWS como cloud provider, para garantir que nossas soluções sejam escaláveis, seguras e eficientes.
RESPONSABILIDADES E ATRIBUIÇÕES
Desenvolver soluções Full Stack com Node.js no backend e React no frontend.Implementar microserviços utilizando AWS e garantir o uso eficiente das suas ferramentas.Participar do desenvolvimento de pipelines de CI/CD com SST e AWS (CodePipeline, CloudFormation).Colaborar com o time para criar interfaces modernas com Shadcn UI e garantir que os sistemas sejam escaláveis e seguros.
