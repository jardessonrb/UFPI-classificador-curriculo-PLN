RESPONSABILIDADES E ATRIBUIÇÕES
We are looking for a Senior Full Stack Developer to contribute to the development of a Next.js-based application. The candidate will be responsible for implementing new features on the front end, creating APIs on the back end, and optimizing the CI/CD infrastructure. The project uses mock data, automated testing with Jest, Storybook for component documentation, and GitHub Actions for deployment automation.
**Responsibilities:**
- Develop and maintain front-end components in Next.js.
- Create and integrate RESTful APIs on the back end using Node.js.
- Implement and maintain CI/CD pipelines using GitHub Actions.
- Manage configuration and infrastructure with Docker.
- Collaborate with DevOps and infrastructure teams to improve the development environment.
- Write detailed documentation for new features and system changes.
REQUISITOS E QUALIFICAÇÕES
**Requirements:**
- Extensive experience with Next.js, React, and Node.js.
- Solid knowledge of relational and non-relational databases (MySQL, MongoDB).
- Experience with Docker and CI/CD tools (GitHub Actions).
- Skills in infrastructure setup with Docker and docker-compose.
- Familiarity with automated testing (Jest) and component documentation (Storybook).
- Intermediate or advanced English for written communication and technical documentation.
