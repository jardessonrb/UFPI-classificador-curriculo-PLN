RESPONSABILIDADES E ATRIBUIÇÕES
Responsabilidades:
Desarrollar y mantener el lado del servidor de aplicaciones web y móviles
Colaborar con desarrolladores front-end para integrar componentes del lado del cliente
Implementar lógica de negocio y funcionalidades en el back-end
Diseñar y optimizar bases de datos
Crear y mantener APIs
Realizar pruebas y depuración de código
Participar en revisiones de código y seguir las mejores prácticas de desarrollo
Apoyar en la resolución de problemas técnicos
REQUISITOS E QUALIFICAÇÕES
Conocimientos y experiencia:
Conocimientos básicos de al menos un lenguaje de programación back-end como Python, Java, PHP o JavaScript (Node.js)
Familiaridad con bases de datos relacionales (MySQL, PostgreSQL) y no relacionales (MongoDB)
Comprensión de principios básicos de diseño de APIs RESTful
Conocimientos de control de versiones con Git
Familiaridad con frameworks web como Django, Spring, Laravel o Express.js
Nociones básicas de seguridad web y protección de datos
Conocimientos básicos de arquitecturas de aplicaciones y patrones de diseño
Conocimiento de tecnologías back end como Node.js o PHP.
Familiaridad con metodologías de desarrollo ágiles.
