RESPONSABILIDADES E ATRIBUIÇÕES
Atividades:
● Levantamento e análise de requisitos junto área de negócio;
● Estruturar dados para relatórios e indicadores;
● Desenvolvimento de processos de carga de dados;
● Tuning em Queries e processos existentes;
● Desenvolvimento de estudos e análises de dados buscando melhorias e
eficiência;
● Criação de visões em looker
Conhecimentos Técnicos:
● Ferramentas de BI: Proficiência na ferramenta LOOKER e experiência em
desenvolvimento e otimização de dashboards interativos.
● Linguagens de Programação: Domínio em SQL para consulta e manipulação de
dados. Conhecimento em linguagens como Python para análises estatísticas
avançadas e automação de processos de dados.
● Banco de Dados: Experiência com bancos de dados relacionais (ex.: MySQL,
SQL Server, PostgreSQL). Capacidade de modelagem de dados e
desenvolvimento de processos ETL (Extração, Transformação e Carga de
dados).
● Data Warehousing: Experiência em arquitetura de data warehouse, incluindo a
criação e manutenção de tabelas e esquemas dimensionais.
● Excel Avançado: Habilidade para realizar análises complexas, incluindo o uso de
fórmulas avançadas, tabelas dinâmicas e gráficos.
