DESCRIÇÃO DA VAGA
Ei, tudo bem?
Se você é uma pessoa flexível, curiosa, com perfil analítico relacionado à estratégia, focada em trazer sempre melhorias para o negócio, então, esta vaga é para você!
RESPONSABILIDADES E ATRIBUIÇÕES
Você irá trabalhar com:
Projetar e implementar pipelines de dados para coleta, processamento e armazenamento de dados em larga escala;
Desenvolver e manter a infraestrutura de armazenamento de dados, garantindo alta disponibilidade e desempenho;
Colaborar com equipes multidisciplinares para entender os requisitos de dados e oferecer soluções técnicas eficazes;
Realizar a limpeza, transformação e preparação de dados para análise;
Garantir a segurança e conformidade dos dados, implementando práticas recomendadas;
Monitorar e otimizar o desempenho dos pipelines de dados;
Resolver problemas técnicos relacionados a dados e participar na resolução de incidentes.
REQUISITOS E QUALIFICAÇÕES
O que nós esperamos de você?
Mínimo 2 anos de experiência em Engenharia de Dados;
Experiência com desenvolvimento e implementação de soluções de big data;
Vivência com ferramentas e tecnologias de engenharia de dados, Airflow, Spark, Kafka, etc.;
Atuação com arquiteturas de dados distribuídas;
Experiência com bancos de dados relacionais e não relacionais;
Experiência com ferramentas e tecnologias de machine learning, como TensorFlow, PyTorch, scikit-learn, etc.
Vai ser muito legal se, além disso, você tiver:
Familiaridade com serviços de nuvem, como AWS e a capacidade de projetar soluções de engenharia de dados na nuvem;
Streaming de Dados: Experiência em processamento de dados em tempo real usando tecnologias como Kafka, Apache Flink ou AWS Kinesis;
Containerization e Orquestração: Conhecimento em Docker e Kubernetes para implantação e orquestração de aplicativos;
Criação de arquiteturas e recursos em nuvem utilizando Infra as Code.
