DESCRIÇÃO DA VAGA
Como Analista de dados Sênior, você desempenhará um papel de realizar análises usando dados para definir as melhores estratégias junto aos stakeholders responsáveis por fazer a gestão das operações Omnichannel, gerando insights para melhorar a eficiência das lojas físicas e E-commerce, auxiliando na tomada de decisão e participando ativamente de novos projetos.

RESPONSABILIDADES E ATRIBUIÇÕES
